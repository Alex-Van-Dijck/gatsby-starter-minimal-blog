## Coding

Tijdens mijn laatste week heb ik mijn tijd vooral gespendeerd aan het oplossen van een bug die zich voordeed in mijn nieuwe Chime layout.
Het gebeurde namelijk dat de incoming video stream soms in 2 splitste, dit werd deels veroorzaakt door mijn implementatie van de library maar niet volledig.
Uiteindelijk na heel veel zoeken (en ondertussen kleine code improvements maken) besefte ik dat de issue niet compleet lag aan mijn implementatie maar dat Amazon zelf het component niet bepaald responsive had gemaakt.

Na de issue te vinden heb ik even met Louis samengezeten om te bekijken hoe ik best tewerk ging om dit op te lossen, hij kwam met het idee om met een patch file te werken.
Wat dit doet is een bepaalde file in de library overschrijven telkens hij geüpdated wordt, daarnaast zal de library ook een warning geven wanneer het niet meer mogelijk is om de gedefiniëerde karakters te vervangen.
